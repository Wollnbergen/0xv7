# Sultan Mobile Validator

Run a Sultan validator node on your mobile device (Android/iOS).

## Features

- ✅ Lightweight validator (2s blocks, minimal resources)
- ✅ Background operation support
- ✅ Push notifications for validator events
- ✅ One-tap staking from mobile wallet
- ✅ Real-time rewards tracking
- ✅ Auto-reconnect on network changes

## Requirements

### Android
- Android 8.0+ (API level 26+)
- 2GB RAM minimum
- 5GB storage
- Stable internet connection

### iOS
- iOS 13.0+
- 2GB RAM minimum
- 5GB storage
- Background app refresh enabled

## Installation

### Android APK
```bash
# Download latest release
wget https://releases.sultan.finance/mobile-validator-android-latest.apk

# Install
adb install mobile-validator-android-latest.apk
```

### iOS TestFlight
```
https://testflight.apple.com/join/sultanvalidator
```

## Quick Start

1. **Install the app**
2. **Create or import wallet**
3. **Stake minimum 5,000 SLTN**
4. **Start validator** (one tap)
5. **Earn ~26.67% APY**

## Technical Details

- **Binary size**: ~15MB
- **RAM usage**: 200-500MB
- **Network**: 10-50 KB/s
- **Battery impact**: Minimal (background optimized)
- **Block participation**: Full consensus
