# Sultan Chain - Validator Economics

## Validator Rewards

- **Annual Percentage Yield (APY)**: 13.33%
- **Unbonding Period**: 21 days
- **Minimum Stake**: 100,000 SLTN
- **Slashing**: 1% for extended downtime

## Economics Model

- 4% annual inflation funds all rewards
- No mobile validator bonus (single tier system)
- All validators earn the same 13.33% APY
- Sustainable long-term economics

## Validator Requirements

- Hardware: 8 CPU, 32GB RAM, 1TB SSD
- Network: 1 Gbps connection
- Uptime: 99.9% required
- Stake: 100,000 SLTN minimum

## Earnings Example

With 100,000 SLTN staked:
- Annual earnings: 26,670 SLTN
- Daily earnings: ~73 SLTN
- Monthly earnings: ~2,222 SLTN
