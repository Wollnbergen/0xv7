# Sultan Chain SDK Documentation

## Overview

The Sultan Chain SDK provides developers with tools to build applications on the Sultan Chain network.

## Access

The SDK is available through:
- Official website (coming soon)
- NPM package (after mainnet launch)
- Direct API integration

## Features

- Transaction building and signing
- Smart contract deployment
- Bridge interactions
- Validator queries
- Zero gas fee transactions

## Getting Started

1. Register as a developer on the official website
2. Obtain API keys
3. Install the SDK
4. Connect to Sultan Chain RPC endpoints

## Support

Developer support available through official channels only.
