# Sultan Chain Validator Staking Mechanics

## Requirements
- Minimum stake: 5,000 SLTN
- Maximum validators: 100
- Unbonding period: 21 days

## Rewards
- Base APY: 13.33%
- Distributed every block
- Auto-compound available

## Slashing Conditions
1. Double signing: 5% slash
2. Downtime (>24h): 0.1% slash
3. Invalid blocks: 1% slash
