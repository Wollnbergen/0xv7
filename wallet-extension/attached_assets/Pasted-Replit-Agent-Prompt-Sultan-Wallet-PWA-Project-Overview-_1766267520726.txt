Replit Agent Prompt - Sultan Wallet PWA
Project Overview

Deploy Sultan Wallet - a React PWA for the Sultan blockchain. Sultan is a Native Rust L1 blockchain (NOT Cosmos, NOT Tendermint, NOT Substrate). No smart contracts - all features are protocol-native.

Tech Stack

React 18 + TypeScript
Vite 6 (bundler)
React Router 6
@noble/ed25519, @noble/hashes (cryptography)
@scure/bip39 (mnemonic)
bech32 (address encoding)
PWA with service worker

Key Architecture Rules

Sultan addresses start with sultan1 (bech32 format)
Decimals: 9 (1 SLTN = 1,000,000,000 base units)
Send screen: Only accepts sultan1... addresses (wallet is Sultan-only)
Wrapped assets: sBTC, sETH, sSOL, sTON come from bridges (future Bridge UI)
Staking APY: 13.33%
Min stake to become validator: 10,000 SLTN
Governance deposit: 1,000 SLTN to create proposal
RPC endpoint: https://rpc.sltn.io

Security Features Implemented

Mnemonic encrypted in memory (XOR SecureString)
Private keys derived on-demand, never cached
PIN with SHA-256 hashing
Rate limiting (5 failed attempts → 5 min lockout)
Session timeout (15 min inactivity → auto-lock)
Optional 2FA (TOTP)

To Change Design Globally

Just edit index.css - all screens use CSS variables from there. Example:

Change --color-primary to change the accent color everywhere
Change --border-radius to make buttons more/less rounded
Change --glow-primary to adjust the glow effect
