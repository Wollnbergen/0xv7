# Sultan Chain - Validator Economics

## Validator Rewards

- **Annual Percentage Yield (APY)**: 26.67%
- **Unbonding Period**: 21 days
- **Minimum Stake**: 1,000 SLTN
- **Slashing**: 1% for extended downtime

## Economics Model

- 8% annual inflation funds all rewards
- Single-tier validator system (no mobile bonus)
- All validators earn the same 26.67% APY
- Sustainable long-term economics

## Validator Requirements

- **Hardware**: 8 CPU cores, 32GB RAM, 1TB SSD
- **Network**: 1 Gbps connection
- **Uptime**: 99.9% required
- **Stake**: 1,000 SLTN minimum

## Earnings Example

With 1,000 SLTN staked:
- Annual earnings: 266.7 SLTN
- Daily earnings: ~0.73 SLTN
- Monthly earnings: ~22.22 SLTN

## Why 1,000 SLTN Minimum?

- Low barrier to entry for decentralization
- Accessible to more validators
- Promotes network security through distribution
- Still meaningful enough to ensure commitment

## Proprietary Access Model

Sultan Chain is proprietary technology:
- Core protocol is NOT open source
- SDK available through official website
- RPC endpoints for registered developers
- Controlled third-party access
